Calculator Web App

A simple, elegant, and fully functional JavaScript Calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations and includes features like delete, clear, percentage, and plus/minus.
Features

✔ Basic arithmetic operations: Add, Subtract, Multiply, Divide

✔ AC (All Clear) and DEL (Backspace) functionality

✔ Plus/Minus (±) toggle

✔ Percentage (%) operation

✔ Responsive and minimal design

✔ Clean UI with modern gradient background

Technologies Used

HTML

CSS

JavaScript
